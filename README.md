# Cracked
Fun chicken-themed arcade-style iOS game!

# A chaotic, endless kitchen game where you have to fry as many eggs as you can!

  -  Game is compatible with iPad and iPhone 

  - Users have a seamless experience of cracking eggs and swiping feathers </br>
